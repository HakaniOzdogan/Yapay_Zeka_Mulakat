# AI Interview Coach

Lightweight open-source AI Interview Coach — real-time coaching and offline analysis.

Quick start (development):

1. Build and run everything with Docker Compose:

```bash
docker compose -f docker/docker-compose.yml up --build
```

Services:
- Backend: http://localhost:5000
- Frontend: http://localhost:5173
- Speech service: http://localhost:8000

See `docs/` for architecture and privacy notes.
